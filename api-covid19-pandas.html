Vamos a estudiar los datos disponibles en estas webs, de las que sacaremos todos los datos para las tablas y los gráficos acerca del COVID19 en varios países.

Instalación pandas¶

La instalación de pandas sirve para escribir datos en CSV, Excel, HDF5 y para poder crear, seleccionar y filtrar las tablas de datos. En este caso nos centramos en el estudio de la API de Covid-19, con el fin de poder obtener los datos acerca del desarrollo de la pandemia en diferentes países.
